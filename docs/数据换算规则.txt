读出的数据是由2个寄存器(每个寄存器是16位数据)，共同表达的32位有符号整型数，
对于C程序来说，无需自己写程序换算，将这个值直接存进"有符号32位整型变量"里即可，程序会自动识别成正确的数值


例如：
int32_t aa = 0X00008329(此时aa 等于 33577)
int32_t aa = 0XFFFF7CFD(此时aa 等于 -33539)


如果上述不好理解，也可如下换算：
d = -(0XFFFFFFFF - code + 1)

例如：	code = 0XFFFF7CFD = 4294933757
	0XFFFFFFFF = 4294967295
	d = -(4294967295 - 4294933757 + 1) = -33539
